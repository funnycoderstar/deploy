# promise

const promise = new Promise((resolve, reject) => {
  console.log(1)
  resolve()
  console.log(2)
})

promise.then(() => {
  console.log(3)
})

console.log(4)

 // 1243

Promise的构造函数是同步执行的,then方法是异步的

const promise = new Promise((resolve, reject) => {
  console.log(1);
  resolve(5);
  console.log(2);
}).then(val => {
  console.log(val);
});

promise.then(() => {
  console.log(3);
});

console.log(4);

setTimeout(function() {
  console.log(6);
});
// 1, 2, 4, 5, 3, 6

promise new的时候会立即执行里面的代码, then是微任务, 会在本次任务执行完的时候执行, setTimeout是宏任务, 会在下次任务执行的时候执行
